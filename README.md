# Overview
This repository if for fun things with python and other technologies.
- integrated with GitHub Actions to build and publish Docker Image;

---
### What I can try here:
> `pipenv` required
- install packages:
```bash
make install
```
- run app locally:
```bash
make run
```
- build Docker image:
```bash
make build-docker
```
